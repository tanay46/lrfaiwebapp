La Rotonde

This web-app uses flask and mongohq

To set up flask:

http://flask.pocoo.org/docs/

sudo pip install virtualenv
cd into project
virtualenv venv

And then activate the venv everytime you work on the app: 
. venv/bin/activate

pip install Flask

To run the app, use:
python hello.py
